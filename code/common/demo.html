Ready to use UI components

When designing a website, you come across many UI elements that you want to customize and create lots of different function to work around them, like taking the data out, doing different things on different events, putting data in, changing its styles etc.
webUI is just what you need to keep all the UIs under control, workaround customized UI elements with ease,

In this Demo you will find the UI emements that webUI offers.
webUI has 2 parts-

The static css class defs like Buttons,font styles,Pannels etc.
You can use it like any other normal .css file, It contains some beautifully designed UI elements ready to use and a Responsive Grid system that is adopted from Bootstrap that uses modern wrappers like Grid and Flex styled Display.

Second Part is the JS code that contains a great no of useful functions that can be used to create these UI elements from code based on the JSON you will provide. These not only creates the visual UI components but also defines a whole range of methods that usually you and most of us will be looking for after we create these elements.
